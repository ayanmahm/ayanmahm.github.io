How to customise your Notion!

There are different methods to customise your Notion page using Notion Enhancer

Download the notion enhancer from the Chrome Store, or the Firefox Store

  • You then need to upload the JSON file by clicking the Profile tab, clicking the Up-arrow button and selecting the JSON file.

If you'd prefer the desktop experience, download the notion enhancer app

  • You then need to upload the JSON file by clicking the Profile tab, clicking the Up-arrow button and selecting the JSON file.
